# Convert a did_priors object to a flat Stan data list

Convert a did_priors object to a flat Stan data list

## Usage

``` r
as_stan_data(priors)
```

## Arguments

- priors:

  A `did_priors` object from
  [`set_priors()`](https://ben18785.github.io/metadid/reference/set_priors.md).

## Value

A named list of scalar hyperparameter values.
