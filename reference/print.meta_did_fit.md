# Print a meta_did_fit object

Displays a brief summary of the population treatment effect posterior.

## Usage

``` r
# S3 method for class 'meta_did_fit'
print(x, prob = 0.9, ...)
```

## Arguments

- x:

  A `meta_did_fit` object.

- prob:

  Width of the credible interval. Default `0.9`.

- ...:

  Ignored.
